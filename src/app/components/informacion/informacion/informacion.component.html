Usuarios especiales

Los usuarios especiales son creados para los servidores públicos, tales como policías, tránsito, bomberos, etc. Esto para poder contar con una reputación del 100% de forma permanente, haciendo valer más los reportes que estos usuarios realizan. Para registrarse como usuario especial se deben completar los datos personales y enviar una imagen que compruebe que se es un servidor público dentro del siguiente formulario.
